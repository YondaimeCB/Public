# Installation
```shell
pip install virtualenv
```


# Configuration
```shell
mkdir virtual-demo
cd virtual-demo

python3 -m venv env   #start a virtual envinronment

source env/bin/activate  #activate virtual environment

python3   #enter to python terminal

which python3  #which directory

pip freeze | less   #list modules installed

deactivate   #to deactivate virtual environment
```
